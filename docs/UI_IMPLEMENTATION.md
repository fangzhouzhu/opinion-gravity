# UI 实现说明

## 实现范围
本次已将首页改造为参考图的结构化版本，并保持现有功能可用：
- 登录/注册
- 创建议题
- 进入议题详情参与讨论

## 关键文件
- `apps/web/app/page.tsx`：新首页结构
- `apps/web/components/auth-nav.tsx`：顶部导航与登录态
- `apps/web/app/globals.css`：视觉系统与响应式样式
- `docs/FEATURE_BREAKDOWN.md`：功能拆分文档

## 设计还原要点
1. 信息架构：顶部导航 + Hero + 讨论区 + 右侧栏
2. 视觉语言：浅绿系、圆角卡片、轻边框、柔和渐变背景
3. 数据表达：立场百分比分布条
4. 交互入口：创建议题、进入讨论、登录注册

## 技术说明
- 当前首页数据来自 `GET /v1/topics`
- 搜索框、菜单、分类 chips 目前先按 UI 占位
- 排行榜和灵感卡当前为静态示例数据

2. 引入统一图标库（Lucide）
