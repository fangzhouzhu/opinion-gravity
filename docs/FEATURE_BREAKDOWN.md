# 功能拆分（基于新 UI）

## 1. 顶部导航模块
- 品牌区（观点场 OG）
- 主导航：发现议题 / 热门观点 / 共鸣榜 / 我的关注
- 搜索框（议题与观点）
- 登录态操作：登录/退出/创建议题

## 2. Hero 模块
- 核心价值文案（观点同频）
- 主行动按钮：发起议题
- 次行动按钮：随机进入讨论
- 社区价值点：多元观点、找到共鸣、理性友善
- 视觉增强：同心圆 + 引语卡 + 用户头像

## 3. 正在讨论模块
- 分类筛选 chips（当前为 UI 占位）
- 议题列表卡片：标题、描述、标签、参与人数
- 立场分布条：正方/反方/中立百分比
- 行动入口：进入观点场

## 4. 侧边栏模块
- 共鸣榜：Top5 用户（UI 演示数据）
- 今日灵感卡：价值观提示与社区氛围塑造

## 5. 认证与权限模块
- 注册、登录、退出
- `Bearer Token` 存储在 `localStorage`
- 创建议题/发观点/投票需要登录

## 6. 下一阶段扩展
1. 搜索与筛选联动到 API
2. 热门观点/共鸣榜改为真实数据
3. 我的关注（关注议题、关注用户）
4. 随机讨论入口接真实随机算法
